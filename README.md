# AI/ML Workload Performance Analyzer (Python)

This project simulates AI/ML workloads, specifically matrix multiplication operations, and compares execution performance using a naive CPU approach versus an accelerated NumPy vectorized approach. It demonstrates understanding of AI workload optimization and basic concepts of performance acceleration in computing systems.

# Features
- Naive CPU implementation of matrix multiplication
- Accelerated implementation using NumPy vectorization
- Measures execution time for different matrix sizes
- Visualizes performance difference with a plot
- Demonstrates system optimization, AI acceleration, and computer architecture concepts

# How to Run
1. Make sure Python 3.x is installed on your system.
2. Install required packages:   pip install numpy matplotlib
